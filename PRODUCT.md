## Goal

Reduce the time and effort required to plan meals and shop for groceries while preserving the discovery, creativity, and satisfaction of cooking.

---

## User Stories

### Lists

* As a user, I want to create and manage lists so that I know what needs to be purchased.
  * Grocery lists
  * Bulk purchase lists (Costco, Sam's Club, etc.)
  * Home improvement lists
  * General-purpose lists
* As a user, I want intelligent item suggestions based on my purchase history so that I can build lists faster.
* As a user, I want to share lists with members of my household so that we can collaborate on purchases.

### Recipes

* As a user, I want to create and manage recipes so that I can plan meals.
* As a user, I want to share recipes with members of my household so that everyone can access them.
* As a user, I want to create a recipe from a block of text so that I can quickly capture recipes from memory, notes, or websites.

### Meal Planning

* As a user, I want to schedule meals on a calendar so that I can plan my week.
* As a user, I want my grocery list to be automatically generated from my meal plan so that I know exactly what ingredients I need.

### Discovery

* As a user, I want new recipe recommendations so that cooking remains interesting and enjoyable.
* As a user, I want to control how frequently new recipes are suggested so that recommendations match my preferences.

### Cooking Assistance

* As a user, I want to chat with a Chef Agent so that I can get help with cooking techniques, ingredient substitutions, meal planning, and recipe questions.

---

## Weekly Workflow

1. User opens the app to plan the upcoming week.
2. The app presents a proposed meal plan:
   * Primarily based on the user's saved recipes.
   * Includes a configurable number of new recipe recommendations.
3. User adjusts the plan:
   * Add/remove meals.
   * Swap recipes.
   * Assign meals to specific days.
4. The app generates a grocery list:
   * Required ingredients from the meal plan.
   * Frequently purchased staple items based on purchase history.
5. User reviews and finalizes the grocery list.
6. User shops using the generated list.
7. Throughout the week, the user opens the app to:
   * View the meal plan.
   * Follow recipe instructions.
   * Ask the Chef Agent questions while cooking.
